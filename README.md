This repository contains the **hands-on material** for an Artificial Intelligence course, covering **Deep Learning, Large Language Models, Generative AI, and Reinforcement Learning**.  
The content follows a **bottom-up approach**, from neural network fundamentals to advanced LLMs, diffusion models, and AI agents.

The repository is organized by **chapters**, each one aligned with the course syllabus and taught through practical Jupyter notebooks.

---

## Repository Structure

- **01_NN_Basics**  
  Fundamentals of neural networks, perceptrons, and training basics.

- **02_Images_CNN**  
  Convolutional Neural Networks, image classification, and computer vision foundations.

- **06_Basic_NLP**  
  Classical NLP techniques and early neural approaches to language.

- **07_Advanced_NLP**  
  Transformers, attention mechanisms, and modern NLP pipelines.

- **08_Reinforcement_Learning**  
  Core concepts of RL, environments, agents, and learning through interaction.

- **09_Diffusion_Models**  
  Generative models for images, diffusion processes, and modern generative pipelines.

- **10_Miscellaneous**  
  Supporting notebooks, utilities, and experimental material.

- **11_LLM_BASIC**  
  Large Language Model fundamentals: autoregressive models, prompting, inference.

- **12_LLM_ADVANCED**  
  Fine-tuning, grounding, RAG, agents, multimodality, and advanced LLM applications.
